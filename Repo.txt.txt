# 📧 Email Spam Classifier

A machine learning project to classify emails as **spam** or **ham (not spam)** using Logistic Regression and TF-IDF (Term Frequency–Inverse Document Frequency). The model is trained on a labeled dataset and evaluated using accuracy metrics and visualizations.

---

## 🚀 Features

- Preprocessing and vectorizing text data with `TfidfVectorizer`
- Classification using `LogisticRegression`
- Input-based prediction (custom email text)
- Accuracy score evaluation (train/test split)
- Visualization using `matplotlib` and `seaborn`

---

## 🧪 Libraries Used

- `numpy`
- `pandas`
- `scikit-learn`
- `matplotlib`
- `seaborn`
